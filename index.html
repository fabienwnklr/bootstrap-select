Bootstrap Select

Quickly design and customize responsive mobile-first sites with Bootstrap, the world’s most popular front-end open source toolkit, featuring Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful JavaScript plugins.

Get started

Requirements:

  • Bootstrap v5.2.3+
Install Bootstrap select

Using npm

npm i @fwcie/bootstrap-select

Using yarn

yarn add @fwcie/bootstrap-select

Using pnpm

pnpm i @fwcie/bootstrap-select

Usage

Via data-bss attribute

<select data-bss>
  <option value="1">first</option>
  <option value="2">second</option>
</select>

Via JavaScript

const select = new BootstrapSelect(document.querySelector("#mySelect"))
Get BootstrapSelect instance
const select = new BootstrapSelect(document.querySelector("#mySelect"))["bs-select"]

Examples

Standart select boxe

Make this:

Become this:

<select data-bss>
  <option value="1">first</option>
  <option value="2">second</option>
</select>
Select boxes with optgroups
<select data-bss>
  <option value="1">first</option>
  <option value="2">second</option>
  <optgroup label="First group">
    <option value="g1">first in group</option>
    <option value="g2">second in group</option>
  </optgroup>
</select>
Multiple select boxes

Simple multiple select

<select multiple data-bss>
  <option value="1">first</option>
  <option value="2">second</option>
  <optgroup label="First group">
    <option value="g1">first in group</option>
    <option value="g2">second in group</option>
  </optgroup>
</select>

Multiple select with remove button using data-attribute

<select multiple data-bss data-bss-done-button>
  <option value="1">first</option>
  <option value="2">second</option>
<optgroup label="First group">
  <option value="g1">first in group</option>
  <option value="g2">second in group</option>
</optgroup>
</select>

Multiple select with remove button using JavaScript

<select multiple id="doneBtn">
  <option value="1">first</option>
  <option value="2">second</option>
  <optgroup label="First group">
    <option value="g1">first in group</option>
    <option value="g2">second in group</option>
  </optgroup>
</select>
const select = new BootstrapSelect(document.getElementById("#doneBtn", { doneButton: true })
